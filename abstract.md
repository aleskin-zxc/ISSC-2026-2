# Создание базы данных рудных минералов для обучения прогнозных ML-моделей

В современной минералогии существуют базы данных, основанные на визуализации данных, например dataset MineralImage5k (https://github.com/dataset-ninja/mineral-image-5k), сделанная Sber AI, Fersman Mineralogical Museum, где было добавлено пять тысяч фотографий минералов (описано в этой статье [[2023.MineralsImage5k.A benchmark for zero-shot raw mineral visual recognition and description.Nesteruk S. and etc.OpenSource.pdf]]), но проблема в том, что существуют непрозрачные минералы, которые трудно различить визуально. Также существуют базы данных IMA, которые описывают только структурные, химические свойства и позицию в классификации, что недостаточно для обучения ML (ссылка, откуда была скачана dataset IMA https://www.rruff.net/ima-mineral-list/?). В основном минералогические базы данных предоставлены в аналоговом формате, что усложняет сборку данных dataset для ML. Такие базы данных структурированы либо в алфавитном порядке, либо на основе химической классификации, что не подходит для обучения под ML-модели. Для этой работы были взяты только рудные минералы, т.к. они представляют наибольших интерес для производства в мире, но большинство рудных минералов непрозрачны и для них нельзя померить преломление и поглощение света. На помощь приходит ключевое для рудных минералов свойство - **отражение**. Шумская классифицировала рудные минералы по спектральным кривым отражения (СКО) в спектральной области между 420-720 нм через 20 нм (15 точек), в которых описаны микротвердости и параметры цвета минералов. В определителе приводится атлас для каждой кривой, где они разделены по группам [[1985.Шумская Н. И. Определитель рудных минералов по спектральным кривым отражения.SUMMARY- Л. Недра - 231 с.]]. Такие количественные данные и графики хорошо подходят для ML-моделей и будут ключевыми колонками в нашей базе данных для рудных минералов.
